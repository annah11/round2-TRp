## User Stories â€” Autonomous Influencer

### As an Influencer Agent (Orchestrator)

- I need to translate campaign goals into scheduled task streams so the system can operate continuously.
- I need tasks to include required MCP tools/resources to ensure deterministic execution.

### As a Trend Analyst

- I need to fetch and rank emerging trends so content is aligned with current attention.
- I need relevance scoring with thresholds to avoid off-brand topics.

### As a Content Generator

- I need structured briefs and constraints so I can create consistent posts across channels.
- I need to tag every draft with risk labels and confidence scores.

### As a Community Manager

- I need to triage mentions and comments into reply tasks based on urgency and sentiment.
- I need disclosure and safety checks applied to replies automatically.

### As a Commerce Manager

- I need to propose offers and affiliate placements that align with brand rules.
- I need transactions to be approved before any side-effecting action occurs.

### As a Compliance Judge

- I need to enforce policy, legal, and platform rules before publication.
- I need clear audit trails for decisions and overrides.

### As a Human Reviewer

- I need a simple approve/reject workflow for flagged content and transactions.
- I need to see the reasoning and evidence used to produce outputs.
